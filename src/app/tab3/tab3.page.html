¿Que es este proyecto?

El programa que usted esta utilizando ahora mismo es un aplicativo que utiliza el poder del IoT (Internet of Things), que sirve para poder controlar dispositivos inteligentes que realizaran acciones tales como controlar corriente o tomar datos del ambiente

¿Que objetivo tiene este proyecto?

Este programa quiere mostar al usuario las capacidades del IoT en con ejemplos sencillos, tales como controlar una bombilla, medir la temperatura o sentir movimiento, aunque el aplicativo actual es bastante sencillo, uno se puede hacer una idea de sus posibilidades. Todo esto se creo con la idea de controlar un ambiente del SENA y poder controlar todo con un simple movil

¿Que se utilizo para este proyecto?

Para la programacion de este aplicativo se utilizaron varios servicios y software, los cuales son:

  • Ionic

    • Framework de Angular para la creacion de aplicaciones moviles, con la cual este aplicativo es posible

  • Firebase

    • Base de datos en tiempo real NOSQL, nuestra base de datos y API estan basados en el servicion gratuito de google, que nos permite trabajar con datos en tiempo real

  • Node-Red

    • Servicio open-source y gratuito que sirve para poder pasar datos y acciones de un dispositivo a otro, mediante la creacion de un servidor inalambrico

  • Raspbery PI y ESP32

    • Componentes fisicos del proyecto, son los encargados de funcionar como servidor(Raspberry) y como receptor de informacion y ejecutor de acciones(ESP32)
